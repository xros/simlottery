simlottery
==========
    By Alexander Liu


#### What's inside?

* A lottery simulater

#### How to use?

* Try `python debug.py`. then you will get something like this:

    ```
    #### Lottery generator ####
    nubmer amount: 6, number range: 1,49 (49 not included), unique: False, sort: True

    [6, 8, 12, 31, 36, 49]
    #### Lottery generator ####
    nubmer amount: 4, number range: 1,6 (6 not included), unique: True, sort: False

    [2, 3, 5, 6]
    ```
